Sistema de Gestión de Estudiantes

Aplicación académica desarrollada en Java con JavaFX y MySQL, que permite gestionar estudiantes mediante un login seguro, roles diferenciados y operaciones CRUD completas.

Objetivos

Objetivo General
Construir un sistema académico que permita gestionar estudiantes con seguridad y control de acceso.

Objetivos Específicos
- Implementar un login con contraseñas encriptadas.
- Definir roles de usuario con permisos distintos.
- Desarrollar un CRUD completo para estudiantes.
- Garantizar persistencia de datos en MySQL.

Características Principales
- Login con contraseñas encriptadas.
- Roles de usuario: administrador, estándar e invitado.
- CRUD completo de estudiantes.
- Persistencia de datos en MySQL mediante JDBC.
- Interfaz gráfica construida con JavaFX y FXML.


Requisitos del Sistema
- Software necesario:
  - JDK 17 o superior
  - MySQL Server 8.0
  - IDE recomendado: NetBeans, IntelliJ IDEA o VSCode

Instalación
1. Clonar el repositorio:

   git clone https://github.com/Marlon6115/PROYECTO-POO.git
   
2. Importar el proyecto en el IDE.
3. Configurar las librerías de JavaFX y JDBC.
4. Crear la base de datos en MySQL.
5. Ejecutar el script SQL incluido para generar las tablas.
